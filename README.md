
# Ejemplo proyecto final

Proyecto final de Analisis

## Run Locally

Clone the project

```bash
  git clone https://github.com/Lejoser40/ProyectoDise-oDeSistemas.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
